## 📌 Project Overview

This project applies the full machine learning workflow to a real-world supervised classification problem. Using the **UCI Adult Income Dataset**, the goal is to predict whether an individual's annual income **exceeds $50,000/year** based on demographic and employment attributes drawn from the 1994 U.S. Census Bureau data.

The project covers:
- Exploratory Data Analysis (EDA)
- Data preprocessing and pipeline construction
- Training and evaluating multiple ML models
- Hyperparameter tuning
- Final comparative evaluation and reporting

## 📊 Dataset

**Name:** Adult Income Dataset (a.k.a. Census Income Dataset)  
**Source:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/2/adult) | [Kaggle](https://www.kaggle.com/datasets/uciml/adult-census-income)  
**Original Authors:** Ronny Kohavi & Barry Becker (1996)  
**Rows:** ~48,000 individuals  
**Columns:** 15 (14 features + 1 target)  
**Task Type:** Binary Classification  
**Target Variable:** `income` → `<=50K` (0) or `>50K` (1)

### Features

| Feature | Type | Description |
|---|---|---|
| `age` | Integer | Age of the individual |
| `workclass` | Categorical | Employment sector (Private, Gov, Self-emp, etc.) |
| `fnlwgt` | Integer | Census sampling weight |
| `education` | Categorical | Highest education level attained |
| `educational-num` | Integer | Education level as a number |
| `marital-status` | Categorical | Marital status |
| `occupation` | Categorical | Type of occupation |
| `relationship` | Categorical | Relationship role in household |
| `race` | Categorical | Race of the individual |
| `gender` | Categorical | Gender (Male/Female) |
| `capital-gain` | Integer | Capital gains |
| `capital-loss` | Integer | Capital losses |
| `hours-per-week` | Integer | Average hours worked per week |
| `native-country` | Categorical | Country of origin |
| `income` | **Target** | `<=50K` or `>50K` |

### Class Distribution (Imbalanced)
- `<=50K` (Class 0): ~75% of records
- `>50K` (Class 1): ~25% of records

### Missing Values
The dataset contains missing values represented as `?` in columns: `workclass`, `occupation`, and `native-country`. These are handled during preprocessing using `SimpleImputer`.

## 📝 Quick Note — Jupyter Notebook vs Python File

All solution files are provided as **Jupyter Notebooks (`.ipynb`)** by default, which is the recommended way to run this project as it allows step-by-step execution and inline visualizations.

However, you are free to use **VS Code** or run plain **Python `.py` files** as per your preference — the code is fully compatible with both environments.

### 🔄 Convert Any Notebook to a .py File

If you prefer to run a plain Python script, convert any notebook with a single command:

```bash
jupyter nbconvert --to script notebooks/01_EDA.ipynb
```

To convert **all notebooks at once:**
```bash
jupyter nbconvert --to script notebooks/*.ipynb
```

This will generate a `.py` file in the same folder as the notebook, which can then be run with:
```bash
python notebooks/01_EDA.py
```

> ⚠️ Note: After conversion, matplotlib plots will not display inline — add `plt.show()` at the end of each plot block if running as a `.py` file.
